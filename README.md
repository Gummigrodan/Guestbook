# Guestbook
Made by: Isac Larsson (2026-03-27)

# What it does
Its a guestbook application that allows my guests to 
write reviews about their visit. You can write your name
and your message.

# How it works
I made it with a bunch of javascript code that listens to
the event "submit" to create a post. Then it gets stringified
and stored in a json file called "messages". Unfortually you
cant visit the website because github doesnt support full-
stack websites. But i made a video file to showcase how it works.

https://github.com/user-attachments/assets/2c5c70d3-8ad7-4c40-9a26-6020ca96029d
